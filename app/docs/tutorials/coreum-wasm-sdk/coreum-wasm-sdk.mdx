Coreum WASM SDK contains Rust data types to be used in WASM smart contracts interacting with the Coreum blockchain.
They allow developers to issue and execute on-chain messages managing Smart Tokens.

## Proto types

The proto types are generated using <a href="https://github.com/CoreumFoundation/coreum-rust-protobuf" class="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold" target="_blank" rel="noopener noreferrer">coreum-rust-protobuf</a>
 according to our current version of the chain to make them compatible with tools like our <a href="https://github.com/CoreumFoundation/test-tube" class="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold" target="_blank" rel="noopener noreferrer">coreum-test-tube</a>
 or to interact with our chain using gRPC and Rust, they should not be manually modified. Instructions on how to use the tool are in <a href="https://github.com/CoreumFoundation/coreum-rust-protobuf" class="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold" target="_blank" rel="noopener noreferrer">coreum-rust-protobuf</a>
 and can be used to generate any Rust protobuf types for any version of our chain.
