# Faucet

You can fund your account here.

If you don't have an account, there are three ways to generate it:

-   (for all users) Go to <a href="/docs/tools/wallets" target="_blank" className="text-[#24cc8e] hover:text-[#1db07a] text-lg font-bold">wallet</a>
 page, connect(install) wallet and create account
    there;
-   (for advanced users) Click `Generate Funded Wallet` button below;
-   (for advanced users) Or run `cored` CLI command:

```
export COREUM_CHAIN_ID="{Chain ID}"
cored keys add <name-of-the-key> --chain-id=$COREUM_CHAIN_ID
```

`{Chain ID}` depends on the Network, you can find value at
<a href="/docs/tutorials/get-started/setup-cli" target="_blank" className="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold">network variables page</a>


## Mainnet

You can generate your address at <a href="/docs/tools/wallets" className="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold">wallet page</a>
, and you can find the list of Coreum
Markets <a href="https://coinmarketcap.com/currencies/coreum/markets/" target="_blank" rel="noopener noreferrer" className="text-[#24cc8e] hover:text-[#1db07a] text-md font-bold">here</a>


Note: Check if CEX supports withdrawal into Coreum Network beforehand.
