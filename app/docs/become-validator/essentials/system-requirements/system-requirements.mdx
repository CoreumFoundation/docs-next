# System Requirements

### Mainnet

> [!WARNING]
> Use only SSD or NVMe hard drives, directly attached to the motherboard.

Recommended requirement - to use a dedicated bare-metal server

If you use Cloud Computing Instance, ensure that your SSD disk is directly attached to the motherboard.
Otherwise, you might experience IO delay and risk to be jailed of not signing blocks.
